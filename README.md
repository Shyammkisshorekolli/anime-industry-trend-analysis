# Anime Industry Trend Analysis

## Overview
This project performs exploratory data analysis on a real-world anime dataset to study long-term trends in genres, ratings, popularity, and media formats across decades.

## Objectives
- Understand how anime genres have evolved over time
- Analyze trends in audience ratings and popularity
- Study relationships between ratings, episode count, duration, and user engagement

## Key Analysis Performed
- Cleaned and preprocessed data by handling missing values and parsing release dates
- Engineered temporal features such as release year and decade
- Exploded multi-genre entries to enable accurate genre-wise analysis
- Analyzed genre evolution, maturity trends, and popularity growth
- Visualized correlations between ratings, number of episodes, duration, and list users

## Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Jupyter Notebook

## Dataset
The dataset contains information about anime titles including genres, ratings, release dates, episode count, duration, and popularity metrics.
